# Adicionar Espaço
### Caminho principal
1. O utilizador seleciona a opção "Adicionar Espaço" na janela "EasyGarden".
2. O sistema mostra a janela "Registar Espaço".
3. O utilizador preenche o endereço, área e tipo (relvado, canteiros ou misto), seleciona o responsável pelo espaço e seleciona a opção "Aceitar".
4. O sistema guarda os dados e fecha a janela "Dados Espaço".
### Caminhos alternativos
3.1 O utilizador seleciona a opção "Cancelar".
3.1.1 O sistema fecha a janela "Registar Espaço".
3.2 O utilizador não preenche todos os campos obrigatórios.
3.2.1 O sistema mostra a mensagem "Campos obrigatórios por preencher".
3.3 O utilizador preenche o endereço com menos de 5 ou mais de 100 caracteres.
3.3.1 O sistema mostra a mensagem "Endereço inválido".
3.4 O utilizador preenche a área com um valor inferior a 1 ou superior a 1000.
3.4.1 O sistema mostra a mensagem "Área inválida".

#Editar Espaço
### Caminho principal
1- O utilizador seleciona a opção "Editar Espaço" na janela “EasyGarden”.
2- O sistema mostra a janela “Dados Espaço”.
3- O utilizador altera os campos e seleciona a opção "Aceitar".
4- O sistema guarda os dados e fecha a janela “Dados Espaço”.
### Caminhos alternativos
1.1- O utilizador seleciona a opção "Editar Espaço" na janela “EasyGarden” sem um espaço selecionado.
1.1.1 - O sistema mostra a mensagem "Espaço não selecionado".
3.1- O utilizador seleciona a opção "Cancelar".
3.1.1 - O sistema fecha a janela “Dados Espaço”.

# Remover Espaço
### Caminho principal
1- O utilizador seleciona um espaço na janela “EasyGarden” e seleciona a opção "Remover Espaço".
2- O sistema mostra uma janela de confirmação com a mensagem "Tem a certeza que deseja remover o espaço?".
3- O utilizador seleciona a opção "Sim".
4- O sistema remove os dados e fecha a janela de confirmação.
### Caminhos alternativos
1.1- O utilizador seleciona a opção "Remover Espaço" na janela “EasyGarden” sem um espaço selecionado.
1.1.1 - O sistema mostra a mensagem "Espaço não selecionado".
3.1- O utilizador seleciona a opção "Não".
3.1.1 - O sistema fecha a janela de confirmação.

# Ver Resumo dos Serviços
### Caminho principal
1- O utilizador seleciona um espaço na janela “EasyGarden” e seleciona "Ver Serviços".
2- O sistema mostra a janela "Dados dos Serviços" com a lista dos serviços efetuados no espaço (incluindo valores por serviço e total).
### Caminhos alternativos
1.1- O utilizador seleciona a opção " Ver Serviços " na janela “EasyGarden” sem um espaço selecionado.
1.1.1 - O sistema mostra a mensagem "Serviço não selecionado".

# Agendar Serviço
### Caminho principal
1- O utilizador seleciona a opção "Agendar Serviço" na janela "EasyGarden".
2- O sistema mostra a janela "Dados Agendamento".
3- O utilizador preenche a data, o preço previsto, o tipo de serviço (manutenção ou preparação) e seleciona a opção "Aceitar".
4- O sistema guarda os dados e fecha a janela "Dados Agendamento".
### Caminhos alternativos
3.1- O utilizador seleciona a opção "Cancelar".
3.1.1- O sistema fecha a janela "Dados Agendamento".

# Registar Serviço
### Caminho principal
1- O utilizador seleciona um espaço na janela "EasyGarden " e seleciona a opção "Registar Serviço"
2- O sistema mostra a janela “Dados Agendamento” com os dados do serviço agendado para esse espaço.
3- O utilizador seleciona a opção "Aceitar".
4- O sistema guarda os dados e fecha a janela "Dados Agendamento".
### Caminhos alternativos
2.1- Não há espaço selecionado ou o espaço selecionado não tem serviço agendado.
2.1.1- O sistema mostra a mensagem "Não há qualquer serviço agendado ".
3.1- O utilizador seleciona a opção "Cancelar".
3.1.1- O sistema fecha a janela "Dados Agendamento".
3.2 – O utilizador edita os dados do serviço e seleciona a opção “Aceitar”
3.2.1 – O caso de uso continua no ponto 4 do CP
